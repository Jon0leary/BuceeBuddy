"# BuceeBuddy" 
Buc-ee’s Buddy is a web application that tracks Buc-ee’s locations visited by the user using the Google Maps API.
The application allows users to store locations, track purchases, and visualize travel history.

-- Tech Stack --
JavaScript
Node.js
Google Maps API
HTML/CSS
MongoDB

-- Features --
• Map visualization of Buc-ee’s locations
• CRUD operations for visited locations
• Purchase tracking
